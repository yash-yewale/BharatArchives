# BharatArchives
### **📜 BharatArchives - Preserving India's Rich Heritage on Blockchain**  

🚀 **BharatArchives** is a **Web3-powered NFT Marketplace** that secures India's cultural and historical assets on the **Polygon blockchain**. With **authenticated NFTs** of rare manuscripts, ancient artifacts, and historical moments, BharatArchives ensures the **digitization and preservation of India's heritage** for future generations.  

---

## **🌟 Features**  

### **🔹 1. Explore India's Historical NFTs**  
- **Categories:** Artifacts, Manuscripts, Independence, Indian Kings, Wars, Rare Paintings, and more.  
- **Detailed NFT Pages:** Each NFT has an authenticity proof, description, historical significance, and blockchain verification.  
- **High-end UI/UX:** Professional, animated, and museum-like interface with modern aesthetics.  

### **🔹 2. Web3 & Blockchain Integration**  
- **Blockchain:** NFTs are stored on the **Polygon (MATIC) network** for low-cost, high-speed transactions.  
- **NFT Wallet Integration:** Users can connect **MetaMask** to buy, sell, and manage NFTs.  
- **Dynamic Pricing Algorithm:** NFT prices fluctuate based on **demand and supply** like a stock market.  

### **🔹 3. Buy & Resell NFTs**  
- **Payment Methods:** Purchase via **Crypto (MetaMask)** or **Credit/Debit Cards**.  
- **Resale Marketplace:** Users can list NFTs for resale, set prices, and view market trends.  
- **Wishlist & Portfolio:** Save favorite NFTs and track owned NFTs in the **wallet dashboard**.  

### **🔹 4. Stock-Market Style Price Tracking**  
- **Live NFT Trends:** View **price fluctuation graphs** (5 years, 3 years, 1 year, etc.).  
- **Historical Data:** Track past ownership, transactions, and NFT market demand.  

### **🔹 5. Community & Contribution**  
- **Contribute Historical Data:** Users can submit historical documents or artifacts to be converted into NFTs.  
- **Decentralized & Transparent:** Ensures history remains **immutable** and **accessible** forever.

## **⚡ Tech Stack**  

### **🔹 Frontend (React + Web3)**  
- **React.js** (V18) → Modern UI development  
- **Redux Toolkit** → State management  
- **Web3.js / Ethers.js** → Blockchain interaction  
- **Chart.js** → NFT price tracking graphs  
- **Swiper.js** → Animated NFT sliders  
- **CSS Animations** → Smooth UI transitions  

### **🔹 Backend (Flask + Blockchain)**  
- **Flask** → Lightweight backend framework  
- **IPFS / Pinata** → Decentralized NFT storage  
- **Smart Contracts (Solidity)** → Ethereum-based NFT contracts  
- **Polygon Blockchain** → Low-cost NFT minting

## **🚀 Installation & Setup**  

### **1️⃣ Clone the Repository**  
```sh
git clone https://github.com/yourusername/BharatArchives.git
cd BharatArchives/frontend
```

### **2️⃣ Install Dependencies**  
```sh
npm install  # Install frontend dependencies
```

### **3️⃣ Start the Frontend**  
```sh
npm start
```
📌 Open **http://localhost:3000/** in the browser.  

### **4️⃣ Setup Backend (Optional, if using Flask backend)**  
```sh
cd ../backend
pip install -r requirements.txt
python app.py
```

## **🌍 Connect to Blockchain**  
1. Install **MetaMask** & switch to **Polygon (MATIC) Network**.  
2. Fund your wallet with **test MATIC tokens** (if testing on testnet).  
3. Buy NFTs using **crypto or card payments**.
